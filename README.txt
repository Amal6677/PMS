Project Title:

Pharmacy Management System is a web/mobile application developed
for an online pharmacy store. This application helps to manage
all the sales data ,all the stocks of the product and purchase
details from supplier.

Before Starting:
Pharmacy Management System has backend and frontend part. In this repository 
currently Backend code is only available which consist of 
6 REST microservices created using Spring boot with maven.
Requirement:

Software:
Java- jdk8 or above

IED:
-Spring tool suite or eclipse with spring project support.
